# Tech Stack

## Core
- **JavaScript (Vanilla)**: No framework overhead. Direct DOM/Canvas manipulation.
- **HTML5 Canvas**: High-performance 2D rendering.

## Build & Tooling
- **Vite**: Modern frontend build tool. Fast HMR (Hot Module Replacement).
- **NPM**: Package management.

## Physics
- **Matter.js**: 2D rigid body physics engine. 
  - *Why?* Allows for "juicy" interactions like dragging the pet, throwing food, and realistic collisions.

## Testing
- **Vitest**: Blazing fast unit test framework. Compatible with Vite config.

## Styling
- **CSS3**: Variables and Flexbox/Grid for the UI overlay.
